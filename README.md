# Lighthouse Project

> A client information and check-in system for Lighthouse


## Keys required
- LIGHTHOUSE_SECRET_KEY
- mysql_user
- mysql_password
- mysql_password_root
- [Mailgun](https://www.mailgun.com/) configuration and secret keys

## Build/Deploy
```
./build-app.sh
docker-compose up -d
```
